# Edge Add-ons Store — Listing Copy

## Description

Keep your browser tidy by instantly spotting and closing duplicate tabs.

Duplicate Tab Manager automatically detects when you have the same page open in multiple tabs and gives you a clear, easy way to clean them up without hunting through dozens of open tabs manually.

**Features:**

- **Instant detection** — Monitors your tabs in real time and shows a badge count on the extension icon whenever duplicates are found
- **Visual dot indicator** — Adds a colored dot to the favicon of duplicate tabs so you can spot them at a glance
- **One-click cleanup** — Click the extension icon to see a list of all duplicate groups, then close extras with a single button
- **Click to focus** — Jump directly to any tab from the popup without losing your place
- **Customizable dot color** — Choose from preset colors or pick any custom color for the duplicate indicator
- **Strict or relaxed matching** — Strict mode treats URLs with different fragments (e.g. #section1 vs #section2) as separate tabs; turn it off to treat them as duplicates of the same page
- **Highlights the active tab** — The popup marks which tab is currently open so you always know which one to keep
- **Keyboard shortcut** — Open the popup instantly with Alt+Shift+A (customizable via edge://extensions/shortcuts)

**Privacy:** All processing happens locally in your browser. No data is collected or sent anywhere.
